what is module?
-> when we divide a small program in different modules, this is called modular programming.

default & single exports
module.exports = {add, sub};

--> multiple exports 
exports.add = (a, b)  => {
    return a+b
}
exports.sub = (a, b)  => {
    return a-b
}
